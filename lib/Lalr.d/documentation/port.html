Portability

The program has been successfully tested on a number of Scheme interpreters and compilers, including Gambit v3.0, MzScheme v103.5, MzScheme v205, SISC 1.5, Chicken, Bigloo 2.5a, Kawa 1.7, and Guile 1.6.0.

It should be portable to any Scheme interpreter or compiler supporting low-level, non-hygienic macros à la define-macro. If you port lalr-scm to a new Scheme system and you want this port to be included in the next releases, please send a request at: boucherd@iro.umontreal.ca
